Module 2 Assignment

Instructions

In this assignment, update your current website index page to include the following:

  1. Create an external CSS style sheet (you can update the one attached to this file). In your stylesheet include at least 1 class and 1 id. Feel free to create more than 1 of each if needed.
  2. Tweak your document's font using the font-family, font-size, and color style rules. Don't go overboard here. Choose only one paragraph. No need to make the entire font of your page with all kinds of fonts. You may also use web fonts (see pae 134)
  3. Create content and use the text-align style rule to align your text to the right margin of your document (see figure 6.5 for example)
  4. Create a list using one of the 3 types of HTML lists as shown on page 139. The list can include anything you want to. The list must contain at least 3 items.
  5. Create a table on your index page. Come up with data to place in your table. Use styles to format and align your table properly on the page.
  6. Create content on your index page that is shown in three columns (see figure 6.16)
